So, How's the Weather?

This project was created to answer the question: "What's the weather like as we approach the equator?"

Previous Next

Summary

The purpose of this project was to take the learnings about Python requests, APIs, and JSON and analyze the weather with respect to the equator. To accomplish this, we queried the OpenWeatherMap API to create a representative model of weather across world cities. We used Python script to visualize the weather of these 500+ cities varying in distance from the equator.

Data points were collected around cloudiness, humidity, maximum temperature, and wind speed. This site will display the data, visualizations and conclusions of the efforts.
